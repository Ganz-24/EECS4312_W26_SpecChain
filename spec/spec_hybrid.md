# Requirement ID: FR_hybrid_1
- Description: The system shall allow users to access and start at least one meditation session without requiring payment.
- Source Persona: Value Conscious Free User (P1)
- Traceability: Derived from review group G2
- Acceptance Criteria: If a new user installs and opens the app, the system must allow them to start at least one meditation session without requiring payment.
- Notes: Rewritten from automated requirement to remove vague wording like “basic features” and make access measurable.
